- Custom CSS styling 🎭

## 📖 Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/esraarabee1/TeradevelopersTaskWorkSpace.git
   ```
2. Navigate to the project folder:
   ```bash
   cd TeradevelopersTaskWorkSpace
   ```
3. Open `index.html` in your browser.
